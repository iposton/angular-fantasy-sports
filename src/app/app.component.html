About FSR X

I made this app because I needed better fantasy sports reserch tools and I wanted to learn how to use angular2.

With this app I plan to make it easier to find unique fantasy players stats and rankings with a clean modern user friendly ui.

I will be adding new features to come for example I would like to show the top of a baseball hitting order 1-4 each day and show top hockey lineups by most ice time.

I am using mysportsfeeds.com api to get all the current info I need to build my idea.

This is a Progressive Web App using Angular 10, and nguniversal (server side rendering).

I am a web developer from Los Angeles and I enjoy building fun apps with angular. Thanks for stopping by! ‚ò∫
